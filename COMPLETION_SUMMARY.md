## 🚀 HOW TO USE

### 1. **Start Development Server**
```bash
cd v:\prtotfolio
npm run dev
```
Open: http://localhost:3001

### 2. **Add Your Photo**
Place image at: `public/profile.png` (300×350px)

### 3. **Customize Content**
Edit component files in `src/components/`:
- Update your name, title, experience
- Add your skills and projects
- Update social links and email
- Change colors in `src/index.css`

### 4. **Build for Production**
```bash
npm run build
```

### 5. **Deploy**
- **Vercel**: `vercel` (easiest)
- **Netlify**: Drag & drop `dist` folder
- **GitHub Pages**: Push to `your-username.github.io` repo
- **Traditional Hosting**: Upload `dist` folder via FTP
