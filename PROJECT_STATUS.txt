🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

1. START DEVELOPMENT:
   $ cd v:\prtotfolio
   $ npm run dev
   → Opens at http://localhost:3001

2. ADD YOUR PHOTO:
   → Place image at: public/profile.png (300×350px)

3. CUSTOMIZE CONTENT:
   → Edit src/components/*.jsx files
   → Update your name, skills, projects, contact info

4. CHANGE COLORS (optional):
   → Edit src/index.css
   → Update --primary-color, --secondary-color, --tertiary-color

5. BUILD FOR PRODUCTION:
   $ npm run build
   → Creates dist/ folder (~450KB)

6. DEPLOY (Choose one):
   $ vercel                          (Vercel - easiest)
   Drag dist/ to netlify.com         (Netlify)
   git push to your-username.github.io  (GitHub Pages)

🎨 ANIMATION CUSTOMIZATION
═══════════════════════════════════════════════════════════════════════════════

TYPING SPEED:
  src/components/Hero.jsx, line ~55
  <TypingEffect text="..." speed={60} />
  Lower = faster, Higher = slower

PHOTO FRAME ROTATION:
  src/components/Hero.css
  animation: rotateFrame 25s infinite linear;
  Change 25s to adjust rotation duration

GRADIENT FLOW SPEED:
  src/components/GradientWord.css
  animation: gradientFlow 3s ease infinite;
  Change 3s to adjust gradient animation speed
